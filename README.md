# Site Dashboard - Vibe Coding Starter

A simple product metrics dashboard for the PM Vibe Coding Lunch-and-Learn.

## Getting Started

1. Make sure you have Node.js installed (`node --version`)
2. Install dependencies:
   ```
   npm install
   ```
3. Start the dev server:
   ```
   npm start
   ```
4. Open the URL shown in your terminal (usually http://localhost:5173)

## What's in here

- `index.html` - The dashboard page
- `main.js` - Dashboard logic (filtering, stats, rendering)
- `style.css` - Styling
- `data/sites.json` - Mock site data (15 customer sites across products)

## Exercises

Try asking your AI coding tool to:

- Change the page title and header color
- Add a new column to the table (e.g., page views, region)
- Create a chart showing CSAT scores by product
- Add a "sort by" feature to the table columns
- Highlight rows where CSAT is below 4.0
- Add a new site to the data file and see it appear

## Data File

`data/sites.json` contains 15 mock customer sites with fields like name, owner, product type, plan, CSAT score, monthly revenue, plugins, and page views. Use this data for dashboard exercises in later sessions.
